# helm-chart-squid
Helm Chart for Squid Proxy (based on honestica/lifen-charts)

## Quick Usage

```
$ helm repo add cnpsquid https://chickenandpork.github.io/helm-chart-squid/
"cnpsquid" has been added to your repositories

$ helm repo update
Hang tight while we grab the latest from your chart repositories...
... ...
...Successfully got an update from the "cnpsquid" chart repository
... ...
Update Complete. ⎈Happy Helming!⎈

$ helm search repo squid
NAME          	CHART VERSION	APP VERSION	DESCRIPTION           
cnpsquid/squid	0.4.3        	v0.1.0     	Squid proxy helm chart
```
